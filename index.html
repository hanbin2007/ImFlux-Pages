快速上手

  1. 导入多张图片。
  2. 选择布局：水平拼接 或 垂直拼接。
  3. 如需可选的精确压缩，在导出前设置目标大小（KB/MB）。
  4. 一键合并并保存到相册或文件。
说明：所有处理均在设备本地完成，不会上传到服务器。应用无广告干扰。

系统要求

  • iOS / iPadOS：建议 iOS 16.6 或更高版本。
  • 权限：照片读写（仅用于本地合并与保存）。

常见问题（FAQ）

1) 如何获得近乎无损的结果？

  • 不要反复多次合并同一批图片。
  • 在导出时关闭压缩。

2) 精确压缩后文件大小与目标值不完全一致？

由于图像内容与编码器特性，实际结果会围绕目标值微调（±若干 KB）。如需绝对保守，可稍微提高目标值。

3) 处理速度变慢或内存紧张？

  • 减少单次合并的图片数量或降低输出分辨率。
  • 确保设备有足够的可用存储空间。

4) 目前不支持哪些功能？

  • 暂未提供导出格式选择。
  • 暂无 sRGB 等色域/色彩配置项。
  • 暂无「网格拼图」模式（仅水平/垂直）。

Quick Start

  1. Import multiple images.
  2. Select layout: horizontal or vertical stitching.
  3. For optional precise compression, set a target size (KB/MB) before exporting.
  4. Merge in one tap and save to Photos or Files.
Note: All processing is performed on-device. Nothing is uploaded. No ads.

Requirements

  • iOS / iPadOS: iOS 16.6 or later recommended.
  • Permissions: Photos read-write (local use only).

FAQ

1) How to achieve near-lossless results?

  • Do not merge the same set of images repeatedly.
  • Disable compression when exporting.

2) Final size slightly differs from the target?

Due to image content and encoder behavior, the result converges around the target (± a few KB). Set a slightly higher target if required.

3) Slow or memory-heavy merges?

  • Reduce the number of images per merge or lower the output resolution.
  • Ensure adequate free storage on the device.

4) Features not supported (currently)

  • No export format selector.
  • No color space options such as sRGB.
  • No grid layout (horizontal/vertical only).
